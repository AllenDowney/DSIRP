Outline¶

This book is an introduction to data structures in Python, using a web search engine as a motivating example. It is based in part on Think Data Structures, which uses Java.

This material is a work in progress, so your feedback is welcome. The best way to provide that feedback is to click here and create an issue in this GitHub repository.
